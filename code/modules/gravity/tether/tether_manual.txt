<h2>Indicators</h2>
<p>The front side of the Gravi-Tonne Tether has several indicators that allow for quick visual confirmation of the system status.</p>

<h3>Status Indicator</h3>
<p>The status indicators are located in the top-left area of the main front panel. The indicators can be one of four colors:</p>
<ul>
	<li><b>Green</b> indicates the system is providing gravity.</li>
	<li><b>Yellow</b> indicates the system is not providing gravity.</li>
	<li><b>Red</b> indicates the system is malfunctioning.</li>
	<li><b>Blue</b> indicates the tether is currently processing a shift in gravity.</li>
</ul>

<h3>Charge Indicator</h3>
<p>The charging lights are located in the top-right area of the main front panel.</p>

<h3>Lock Indicator</h3>
<p>
The main input screen on the bottom-right area of the front main panel is the <b>system lock indicator</b>.
If green, the system is unlocked and the tether can be operated. If yellow, the system is locked and no input will be accepted.
</p>

<h3>Intensity Indicator</h3>
<p>The lighting along the left side of the unit displays the current intensity level.
