Hotkeys

Common Keys

  • TAB: Toggle WASD mode on or off
  • F1: Adminhelp
  • F2: Quick screenshot
  • Ctrl+F2: Save screenshot
  • Alt: Examine object clicked on
  • Ctrl: Pull object clicked on
  • Shift: Toggles throw mode on when held down, toggles off when released
  • Ctrl+P: Toggle point mode on or off
  • PgUp: Switch active hand
  • PgDown: Use item in active hand
  • Home: Drop item from active hand
  • End: Toggle throw mode on or off

    • WASD Mode Specific Keys

  • W: Up
  • A: Left
  • S: Down
  • D: Right
  • T: Talk
  • E: Switch active hand
  • C: Use item in active hand
  • Q: Drop item from active hand
  • X: Toggle throw mode on or off
  • P: Toggle point mode on or off
  • B: Toggles point mode on when held down
  • V: Equip item in active hand, if possible
  • G: Bring focus to, and clear, the input bar
  • Z: Move south-west
  • 1: Help intent
  • 2: Disarm intent
  • 3: Grab intent
  • 4: Harm intent
  • R: Flip
  • F: Fart
  • Alt+C: OOC

  • 2: Second tool slot
  • 3: Third tool slot
  • 4: Deselect tool
